# 🌾 KisanMitra

## Run on your desktop — 3 steps

**Step 1 — Install dependencies**
```
pip install -r requirements.txt
```

**Step 2 — Run**
```
python run.py
```

**Step 3 — Open your browser**
```
http://localhost:5000
```

That's it. The database is created automatically on first run.
